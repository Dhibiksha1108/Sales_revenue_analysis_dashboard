# Sales & Revenue Analysis Dashboard

## Project Overview

This Power BI dashboard analyzes sales and revenue performance using the Superstore dataset.

## Features

* Total Sales KPI
* Total Profit KPI
* Total Quantity KPI
* Revenue Trend Analysis
* Top Performing Products
* Category-wise Sales Distribution
* Interactive Region Filter

## Tools Used

* Power BI Desktop
* CSV Dataset

## Outcome

The dashboard provides insights into sales performance, product trends, and category contributions, helping support business decision-making.
